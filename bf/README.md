# 初めに

Small-Step semantics を定義して、それを使ってインタプリタを実行する。
実行の繰り返しには、[2.] がLtac の再帰呼び出しを使っていることに対して、
「do !」 つまり repeat タクティクで行っている。

2個スタックのVMでの実装、すなわち Small-Step semantics は [3.] を参考にした。

Coqのコードは、全体的に [1.] の考え方に沿っている。
